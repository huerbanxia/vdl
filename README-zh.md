## 项目安装

`尽量使用npm命令 ，使用pnpm在打包时会出现找不到依赖的问题`

### 安装

```bash
$ npm install
# 使用 pnpm
$ pnpm install
```

### 启动

```bash
$ npm run dev
# 使用 pnpm
$ pnpm run dev
```

### 构建可分发程序

```bash
# For windows
$ npm run build:win

# For macOS
$ npm run build:mac

# For Linux
$ npm run build:linux
```
